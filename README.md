snakeySnake

       |_ scoreboard.txt
    |_ __init__.py

    |_ game.py
    |_ scoreboard.py

## To run:
- Run `snake-cli` from the command line
- Move using "ASWD" or arrow keys
- Collect points by collecting apples and survivng 
- Game over if the snake runs into itself or any walls
